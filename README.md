# VolcanoPlot-RShinyApp
R Shiny application that allows to create and customize a Volcano plot out of a list of genes with enrichment score (CSV/TSV/TXT/RDS). It is possible to select which gene labels to visualize. Moreover, the Volcano itself, limits, dots and gene labels, can be customized as preferred. The final volcano plot can be downloaded locally (PDF/PNG/SVG).
